# Shell 函数返回值的两种方法

## 1. **echo 方法 - 返回数据**

```bash
add() {
    echo $(( $1 + $2 ))
}
SUM=$(add 10 20)
echo "10 + 20 = $SUM"
```

- **用途：** 返回**实际数据**（字符串、数字等）
- **捕获方式：** 使用命令替换 `$()`
- **返回范围：** 可以返回任意文本内容
- **适用场景：** 需要获取函数的计算结果或输出

### 工作原理
- 函数内部使用 `echo` 输出结果到标准输出（stdout）
- 调用时使用 `$()` 捕获这个输出
- 可以返回任意长度的字符串或多行文本

## 2. **测试表达式/return 方法 - 返回状态码**

```bash
isEven() {
    [ $(($1 % 2)) -eq 0 ]  # 测试表达式，返回退出状态
}
isEven 10 && echo "10 is even"
```

- **用途：** 返回**成功/失败状态**
- **捕获方式：** 使用 `&&`、`||` 或 `if` 判断
- **返回范围：** 0-255（0=成功，非0=失败）
- **适用场景：** 需要判断条件是否满足

### 工作原理
- 测试表达式 `[ ... ]` 会根据条件是否成立返回退出状态
- 退出状态 0 表示成功/真，非 0 表示失败/假
- 可以使用 `&&`、`||` 或 `if` 来根据状态执行后续操作

## 关键区别总结

| 特性 | echo | return/测试表达式 |
|------|------|------------------|
| 返回类型 | 数据（字符串） | 状态码（整数 0-255） |
| 使用方式 | `result=$(func)` | `func && ...` 或 `if func; then` |
| 典型用途 | 计算、数据处理 | 条件判断、验证 |
| 性能 | 需要创建子 shell | 更快，无需子 shell |
| 输出到 | stdout | 退出状态码 |

## 实际应用示例

### Echo 方法 - 数据处理
```bash
# 字符串处理
toUpperCase() {
    echo "$1" | tr '[:lower:]' '[:upper:]'
}
result=$(toUpperCase "hello")
echo "$result"  # 输出: HELLO

# 计算
multiply() {
    echo $(( $1 * $2 ))
}
product=$(multiply 5 6)
echo "5 * 6 = $product"  # 输出: 5 * 6 = 30
```

### 测试表达式方法 - 条件判断
```bash
# 文件检查
fileExists() {
    [ -f "$1" ]
}
fileExists "config.txt" && echo "配置文件存在"

# 数值比较
isPositive() {
    [ "$1" -gt 0 ]
}
if isPositive 10; then
    echo "是正数"
fi
```

## 混合使用

有时可以结合两种方法：

```bash
divide() {
    if [ "$2" -eq 0 ]; then
        return 1  # 返回错误状态
    fi
    echo $(( $1 / $2 ))  # 返回计算结果
}

# 使用方式
if result=$(divide 10 2); then
    echo "结果是: $result"
else
    echo "除法错误"
fi
```

## 注意事项

1. **echo 方法会创建子 shell**，性能略低于 return/测试表达式
2. **return 只能返回 0-255**，超出范围会被截断
3. **测试表达式**实际上没有显式使用 `return`，而是利用了命令的退出状态
4. 如果函数既要返回数据又要返回状态，优先使用 echo 返回数据，通过退出状态判断成功/失败
